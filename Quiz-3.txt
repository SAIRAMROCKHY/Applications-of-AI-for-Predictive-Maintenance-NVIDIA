**Question 1**

Which statement is not true on autocoders?

- A) The output dimension is the same as the input dimension
- B) The encoder encodes its input data into a smaller dimensional space, called the latent space
- C) The decoder network tries to predict the sample class of the input
- D) Autoencoders could be thought of as a generalization of the PCA method

Answer:- C

What is ROC Curve?

- A) A measure of a test's accuracy that considers both the precision and the recall of the test to compute the score
- B) A plot used to indicate imbalances of data distribution
